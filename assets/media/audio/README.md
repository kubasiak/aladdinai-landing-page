# Audio Files

Place your background audio files here.

## Supported Formats:
- MP3 (recommended)
- OGG
- WAV

## File Size:
- Max 10MB per file
- Recommended: 2-5MB for web

## Example:
- `ocean-waves.mp3`
- `ambient-music.mp3`

To add audio:
1. Place MP3 file in this folder
2. Reference in your page code or upload via admin panel (with backend)
